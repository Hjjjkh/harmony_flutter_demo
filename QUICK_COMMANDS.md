# 快速指令参考

## 📝 常用指令模板

### 添加功能

```
添加一个 [功能名称] 页面/组件
添加 [功能描述] 功能
实现 [功能描述]
```

### 修改功能

```
修改 [功能/页面] 的 [具体内容]
更新 [功能/页面] 的 [具体内容]
调整 [功能/页面] 的 [具体内容]
```

### 修复问题

```
修复 [问题描述]
解决 [问题描述]
修复 [页面/功能] 中的 [问题]
```

### 优化代码

```
优化 [代码/功能] 的 [性能/结构/UI]
改进 [代码/功能] 的 [具体方面]
重构 [代码/功能]
```

### 添加原生功能

```
添加调用鸿蒙 [功能名称]
实现鸿蒙 [功能名称] 功能
集成鸿蒙 [功能名称] API
```

### 配置相关

```
配置 [配置项]
设置 [配置项] 为 [值]
更新 [配置文件]
```

```
添加一个登录页面，包含用户名和密码输入框

```
