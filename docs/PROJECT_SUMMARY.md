## 🚀 快速开始

1. **克隆项目**
   ```bash
   git clone https://github.com/Hjjjkh/harmony_flutter_demo.git
   ```

2. **安装依赖**
   ```bash
   flutter pub get
   ```

3. **检查环境**
   ```bash
   .\scripts\check_environment.ps1
   ```

4. **开始开发**
   - 在 Cursor 中打开项目
   - 参考 [开发工作流程](../DEVELOPMENT_WORKFLOW.md)
   - 开始编写代码
