## 📝 如何更新约束规则

### 方式 1: 通过对话框（推荐）

直接在对话框中告诉 AI：

```
更新华为开发约束：[约束名称]
[约束内容]
```

例如：
```
更新华为开发约束：编码规范
添加新的命名规范：...
```

### 方式 2: 手动更新

1. 编辑相应的约束文档文件
2. 添加或修改内容
3. 提交到 Git

### 方式 3: 添加新约束

1. 在 `constraints/` 目录创建新文件
2. 添加约束内容
3. 更新本索引文件

## 🎯 使用方式

AI 在编写代码时会：
1. 参考这些约束规则
2. 遵循编码规范
3. 应用最佳实践
4. 使用推荐的设计模式
