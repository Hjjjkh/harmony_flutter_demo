# 快速投喂文档指南

## 🎯 最简单的方法

**直接在对话框中告诉 AI：**

```
添加华为官方文档：[文档标题]
[文档内容]
```

就这么简单！AI 会自动：
- ✅ 创建文档文件
- ✅ 组织文档结构
- ✅ 更新索引
- ✅ 格式化文档

---

## 📝 实际示例

### 示例 1: 添加开发指南

```
添加华为官方文档：HarmonyOS 应用开发指南

## HarmonyOS 应用开发指南

### 应用架构
HarmonyOS 应用采用分层架构...

### 开发流程
1. 创建项目
2. 配置环境
3. 编写代码
```

### 示例 2: 添加 API 文档

```
添加华为官方文档：HarmonyOS API 参考
链接：https://developer.harmonyos.com/cn/documentation/
内容摘要：包含所有 HarmonyOS 系统 API 的使用说明和示例
```

### 示例 3: 更新约束规则

```
更新华为开发约束：编码规范
添加规范：
- 使用驼峰命名法
- 类名大驼峰
- 变量名小驼峰
```

---

## 📋 文档类型

### 官方文档
```
添加华为官方文档：[标题]
[内容]
```

### 约束规则
```
添加华为开发约束：[规则名称]
[规则内容]
```

### 示例代码
```
添加华为开发示例：[示例名称]
[代码示例]
```

---

## 🔗 添加链接文档

如果文档是外部链接：

```
添加华为官方文档：[文档标题]
链接：[URL]
内容摘要：[摘要]
关键要点：
- 要点 1
- 要点 2
```

---

## ✅ 完成

添加完成后，AI 会：
1. 创建文档文件
2. 更新索引
3. 提交到 Git（可选）

**就这么简单！现在就可以开始添加文档了！** 🚀
