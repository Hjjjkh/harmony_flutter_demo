# 📚 如何投喂文档给 AI

## 🎯 最简单的方法（推荐）

**直接在对话框中告诉 AI：**

```
添加华为官方文档：[文档标题]
[文档内容]
```

就这么简单！AI 会自动处理所有细节。

---

## 📝 实际示例

### 示例 1: 添加开发指南

```
添加华为官方文档：HarmonyOS 应用开发指南

## HarmonyOS 应用开发指南

### 应用架构
HarmonyOS 应用采用分层架构：
- UI 层：使用 ArkUI 框架
- 业务逻辑层：使用 TypeScript/ArkTS
- 数据层：使用分布式数据管理

### 开发流程
1. 创建项目
2. 配置开发环境
3. 编写代码
4. 调试测试
5. 打包发布
```

### 示例 2: 添加链接文档

如果文档是外部链接（如您提到的微信文章），可以这样：

```
添加华为官方文档：HarmonyOS 开发最佳实践
链接：https://mp.weixin.qq.com/s/gLgP7gGU0pmGc2x1hS-0UQ
内容摘要：华为官方发布的 HarmonyOS 开发最佳实践文章
关键要点：
- 应用架构设计
- 性能优化方法
- 开发规范要求
```

**注意**：如果链接需要验证才能访问，您可以：
1. 复制文章内容，直接粘贴给 AI
2. 或者提供文章的关键内容摘要

### 示例 3: 更新约束规则

```
更新华为开发约束：编码规范
添加新的规范：
- 使用驼峰命名法
- 类名使用大驼峰（PascalCase）
- 变量名使用小驼峰（camelCase）
- 常量使用全大写下划线（UPPER_SNAKE_CASE）
```

### 示例 4: 添加示例代码

```
添加华为开发示例：相机功能调用

## 🔗 处理外部链接文档

### 方法 1: 直接复制内容（推荐）

如果链接需要验证，直接复制文章内容：

```
添加华为官方文档：[文章标题]
[粘贴文章完整内容]
```

### 方法 2: 提供摘要和要点

如果无法访问完整内容，提供摘要：

```
添加华为官方文档：[文章标题]
链接：[URL]
内容摘要：[摘要]
关键要点：
- 要点 1
- 要点 2
- 要点 3
```

## 🎯 AI 会做什么

当您告诉 AI 添加文档后，AI 会：

1. ✅ **创建文档文件**
   - 选择合适的目录
   - 使用正确的文件名
   - 使用 UTF-8 编码

2. ✅ **组织文档结构**
   - 格式化 Markdown
   - 添加适当的标题层级
   - 整理代码示例

3. ✅ **更新索引**
   - 更新 `docs/HUAWEI_DOCS/README.md`
   - 添加文档链接
   - 更新目录结构

4. ✅ **提交到 Git**（可选）
   - 如果需要，可以提交到 Git
   - 使用正确的提交信息
